Recommended practices (according to the same Librarians at Princeton) for crafting a file naming convention are:

  • Files names should be consistent

  • Concise yet descriptive

  • Avoid special characters (these will cause headaches when reading into various programs)

  • Include version number

  • Record naming convention in your data management plan

  • Get rid of spaces

    • programs like R do not deal well with spaces - use hyphens, underscores, or capitals to make file names more readable
